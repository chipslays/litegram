Универсальный конструктор клавиатуры.

public static show(array<string|int, mixed>|string $keyboard[, bool $oneTime = false ][, bool $resize = true ][, bool $selective = false ]) : string

Автоматически определяет обычная клавиатура или инлайн. Может залупонькаться на ключ text, например для кнопки запроса гео или контакта. В этом случае лучше юзать markup метод который именно для обычной клавы. TODO: сделать тру универсал и учесть эту херню выше.

Parameters
$keyboard : array<string|int, mixed>|string

Массив с клавиатурой или ключ ранее добалвенной клавиатуры

$oneTime : bool = false

True - показать один раз, False - не прятать клавиатуру после нажатия

$resize : bool = true

True - vаленькая клавиатура, False - большая клавиатура

$selective : bool = false

True - персольная клавиатура для юзера, False - для всех юзеров
